# 4x4x4-ristinolla

Peli, jossa pelaajat asettavat vuorotellen merkkinsä tyhjään ruutuun tavoitteenaan saada neljä omaa merkkiään riviin. Tämän tavoitteen ensin saavuttava pelaaja voittaa pelin.
